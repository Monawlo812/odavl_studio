## Publishing Steps

### 1. Prerequisites

```bash
# Install vsce (VS Code Extension Manager)
npm install -g @vscode/vsce

# Verify publisher account
vsce verify-pat <your-pat>
```

### 2. Package & Publish

```bash
cd apps/vscode-ext

# Package extension (already done)
vsce package

# Publish to marketplace
vsce publish

# Or publish specific version
vsce publish 0.3.0
```
